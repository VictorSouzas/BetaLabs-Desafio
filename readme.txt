Copie a pasta PHP para o diretorio base do seu servidor
altere o arquivo PHP/app/config/app.php insira os dados de acesso ao banco qualquer dúvida acesse https://book.cakephp.org/3.0/en/orm/database-basics.html#configuration
suba o dump.sql no seu banco de dados.
para acessar o sistema use http://localhost/PHP/app/
